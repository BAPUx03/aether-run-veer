# Asset and licence credits

## Original project content

All characters, enemies, platform shapes, collectibles, UI artwork, particles, level layouts, story text, and sound effects are original to **Aether Run: Shardfall**.

- Visuals are drawn at runtime with Phaser Graphics primitives.
- Sound effects are synthesised at runtime with the browser Web Audio API.
- No external sprite, music, sound, model, font, Mario, or Nintendo asset is included.

The seven world background illustrations were generated specifically for this project with OpenAI image generation, then resized and compressed locally for browser performance. They contain no logos, text, franchise characters, or third-party game assets. The Kamal Stepwells and Thar Sky Palace art uses original fictional architecture inspired by broad Indian visual traditions; it does not depict a real sacred site or religious figure.
